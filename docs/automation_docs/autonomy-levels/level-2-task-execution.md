# Level 2: Task Execution

## Description
System executes tasks automatically, human reviews outputs.

## Characteristics
- Automatic task execution
- Human reviews outputs
- Auto-retry on simple failures
- Escalate complex failures

## Use Cases
- Standard development tasks
- Code review automation
- Testing tasks

## Human Involvement
- 60% - System executes
- 40% - Human reviews outputs

## Workflow
1. Planner creates task breakdown
2. Agents execute subtasks
3. Reviewer validates outputs
4. Human approves final result
5. Fixer handles any issues
