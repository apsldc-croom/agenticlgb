# Level 1: Assisted

## Description
System plans tasks, human approves before execution.

## Characteristics
- System generates plans
- Human reviews and approves
- Execution only after approval
- System suggests, human decides

## Use Cases
- Production use with oversight
- Complex multi-step tasks
- High-value operations

## Human Involvement
- 80% - Human approves plans
- 20% - System executes approved plans

## Transition from Level 0
- Add plan generation capability
- Add approval workflow
- Track approval decisions
