# Autonomy Levels

System autonomy levels from manual to self-improving.

## Overview

| Level | Name | Description |
|-------|------|-------------|
| 0 | Manual | Full human control |
| 1 | Assisted | Human approves plans |
| 2 | Task Execution | Execute tasks, human reviews |
| 3 | Phase Execution | Handle edge cases |
| 4 | Roadmap Execution | Handle failures |
| 5 | Self-Improving | Continuous learning |
