# Success Criteria

## Key Metrics

### Task Completion
- **Success Rate**: % of tasks completed successfully
- **First-Attempt Success**: % succeeding without retry/escalation
- **Average Attempts**: Attempts needed per task

### Quality
- **Code Quality**: Linting, testing, review pass rates
- **Output Accuracy**: Correctness of generated outputs
- **Human Approval Rate**: % of outputs approved without changes

### Efficiency
- **Token Efficiency**: Tokens used vs. task complexity
- **Time to Completion**: Wall-clock time for task completion
- **Cost per Task**: Dollar cost per successful task

### Reliability
- **Failure Recovery**: % of failures successfully recovered
- **System Uptime**: Availability of the system
- **Error Rate**: Frequency of errors/exceptions

## Targets (Initial)
- 80% first-attempt success for simple tasks
- 60% first-attempt success for complex tasks
- < 5% error rate
- < $0.50 average cost per task
