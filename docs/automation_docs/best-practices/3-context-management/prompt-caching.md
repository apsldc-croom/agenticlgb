# Use Prompt Caching

Many providers support prompt caching for repeated large contexts.

## Useful For
- Large documentation
- Long codebase
- Repeated system prompts
- RAG systems

## Benefit
You pay less for repeated prompt sections.

## How It Works
Mark sections as cacheable. The API caches repeated content and charges lower rates for cache reads.

## Note
Not all providers support caching. Check your provider's documentation.
