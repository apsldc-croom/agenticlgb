# Hosted API vs Self-Hosting

## Hosted APIs (Use Initially)

Best when:
- Low usage
- Burst usage
- Experimentation
- No GPU management

**Providers:**
- OpenRouter
- Hugging Face Inference
- Together AI
- Novita
- Groq

## Self-Hosting

Best when:
- High volume
- Predictable traffic
- Privacy requirements
- Custom tuning

**But requires:**
- GPU costs
- Infra complexity
- Maintenance

## Recommendation
Start with hosted APIs. Self-hosting usually NOT worth it initially.
