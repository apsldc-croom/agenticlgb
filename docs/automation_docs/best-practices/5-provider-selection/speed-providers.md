# Speed-First Providers

If latency matters, use these providers:

## Fast Providers
- **Groq** - Ultra-fast inference
- **Cerebras** - Extremely fast
- **Fireworks AI** - Fast + good quality

## Best For
- Chat apps
- Autocomplete
- Streaming UX
- Agents
- Real-time applications

## Trade-off
Speed-first providers may have higher token costs but save on wasted time.

## Usage Tip
Use these for user-facing applications where response time matters most.
