# Does the Task Need Reasoning?

This is the biggest optimization decision.

## Low Reasoning Tasks

Use cheap/free fast models.

**Examples:**
- formatting
- summarization
- rewriting
- extraction
- grammar fixes
- SQL formatting
- JSON conversion
- email drafting
- basic CRUD code
- translation
- regex generation
- boilerplate APIs

## Use These for Low Reasoning
- MiniMax
- Qwen
- Llama 3 8B
- Gemma
- DeepSeek small
- Mistral small

These are often enough.

## Key Question
> "How intelligent does this task actually need to be?"

NOT: "What is the best model available?"
