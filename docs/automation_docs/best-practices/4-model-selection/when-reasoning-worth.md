# When Reasoning Models Are Worth It

Use strong reasoning models only for:

- architecture decisions
- debugging hard issues
- multi-step planning
- agent workflows
- compiler/runtime bugs
- advanced math
- research synthesis
- security analysis
- large refactors
- deep code understanding

## Reasoning Models
- Anthropic Claude Sonnet/Opus
- OpenAI GPT-4.x / GPT-5
- Google Gemini 2.5 Pro
- DeepSeek R1
- Qwen reasoning models

## Key Point
Don't waste expensive reasoning models on trivial tasks.
