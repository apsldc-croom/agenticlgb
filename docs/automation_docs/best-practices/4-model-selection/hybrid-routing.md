# Hybrid Routing - Best Real-World Strategy

Smart systems NEVER use one model for everything.

## Typical Production Routing
| Task | Model Type |
|------|------------|
| Rewrite text | cheap small |
| Simple coding | medium |
| Hard debugging | reasoning |
| Final validation | strong model |
| Bulk processing | cheapest possible |

## How It Works
Route requests to appropriate model based on:
- Task complexity
- Required quality
- Time sensitivity
- Cost constraints

## Benefit
This reduces costs massively (50-80%) while maintaining quality where it matters.
