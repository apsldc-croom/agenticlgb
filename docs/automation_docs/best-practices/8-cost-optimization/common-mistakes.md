# Common Mistakes

## Biggest Mistake

Using **GPT-5 / Claude Opus for EVERYTHING**

This burns money unnecessarily.

## Reality

Many tasks can run on:
- MiniMax
- Qwen
- DeepSeek small
- Llama 3

with almost same output quality.

## Wrong Mindset
> "What is the best model available?"

## Right Mindset
> "How intelligent does this task actually need?"

## Other Mistakes

- Not setting max_tokens
- Sending full chat history
- Using reasoning models for formatting
- Not monitoring token usage
- Using large context when not needed
- Ignoring free/open models
