# Planner Agent

Task planning and decomposition.

## Responsibilities

- Understand user requirements
- Break down into subtasks
- Identify dependencies
- Estimate complexity
- Create execution plan

## Inputs
- User task description
- Project context
- Available capabilities

## Outputs
- Task breakdown
- Subtask list
- Dependencies
- Complexity assessment
- Execution order

## Prompts

See [AI Infrastructure: Agent Prompts](../../ai-infrastructure/prompts/agents/)

## Workflow

```
1. Analyze task
2. Identify subtasks
3. Determine dependencies
4. Order tasks
5. Estimate complexity
6. Create plan
```

## Failure Handling

- Ambiguous requirements → Ask clarification
- Unknown domain → Request context
- Complex task → Suggest simpler approach
