# Evaluator Agent

Quality evaluation and metrics.

## Responsibilities

- Evaluate output quality
- Measure performance
- Track metrics
- Identify improvements

## Evaluation Dimensions

| Dimension | Metrics |
|-----------|---------|
| Correctness | Accuracy, completeness |
| Quality | Style, patterns |
| Efficiency | Token usage, time |
| Reliability | Success rate, errors |

## Outputs

- Quality scores
- Metric reports
- Improvement suggestions
