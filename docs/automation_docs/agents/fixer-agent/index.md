# Fixer Agent

Bug fixing and error correction.

## Responsibilities

- Analyze errors
- Identify root cause
- Implement fixes
- Verify solution

## Fix Types

| Type | Description |
|------|-------------|
| Syntax | Fix syntax errors |
| Logic | Fix incorrect behavior |
| Performance | Fix slow code |
| Security | Fix vulnerabilities |

## Workflow

```
1. Analyze error
2. Identify cause
3. Generate fix
4. Test solution
5. Verify success
```

## Retry Logic

- Max 3 attempts
- Escalate if failed
- Log all attempts
