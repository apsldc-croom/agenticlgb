# Tester Agent

Test generation and validation.

## Responsibilities

- Write unit tests
- Create integration tests
- Generate test data
- Verify test coverage

## Test Types

| Type | Purpose |
|------|---------|
| Unit | Test individual functions |
| Integration | Test component interaction |
| E2E | Test full workflows |
| Performance | Test load handling |

## Outputs

- Test files
- Test fixtures
- Coverage report

## Best Practices

- Test edge cases
- Use meaningful names
- Keep tests focused
- Ensure isolation
