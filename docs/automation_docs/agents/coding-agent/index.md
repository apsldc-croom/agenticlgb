# Coding Agent

Code generation and implementation.

## Responsibilities

- Generate code from specifications
- Follow project patterns
- Write clean, maintainable code
- Add appropriate comments

## Inputs
- Task description

- Existing code context

## Outputs

- Updated files
- Test stubs

- Match project style
- Add type hints
- Include docstrings
- Handle errors
- Write testable code
