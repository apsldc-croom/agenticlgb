# Reports

Analysis and summary reports.

## Report Types

| Type | Description |
|------|-------------|
| daily | Daily summary |
| weekly | Weekly analytics |
| cost | Cost breakdown |
| quality | Quality metrics |
| performance | Performance analysis |

## Example Report

```json
{
  "report_id": "uuid",
  "type": "daily",
  "date": "2024-01-15",
  "metrics": {
    "tasks_completed": 50,
    "success_rate": 0.92,
    "total_tokens": 500000,
    "total_cost": 15.00
  }
}
```
