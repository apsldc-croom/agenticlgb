# Naming Conventions

## Prompt Naming

Format: `<domain>-<task>-<version>.md`

Examples:
- `django-api-generator-v3.md`
- `security-review-v2.md`
- `bug-fix-template-v1.md`

## Agent Naming

Format: `<role>-agent.md`

Examples:
- `planner-agent.md`
- `reviewer-agent.md`
- `router-agent.md`

## Config Naming

Format: `<purpose>-config.yaml`

Examples:
- `coding-config.yaml`
- `routing-config.yaml`
- `provider-config.yaml`
